# Login Form

A clean, responsive login form built using HTML, CSS and JavaScript.

![Login Form Screenshot](./Screenshot%202024-07-09%20081113.png)

## Description

This repository contains a simple, modern login form UI demonstrating layout, styling, and basic client-side validation using plain HTML, CSS, and JavaScript. It's intended as a small front-end component you can drop into a static site or use as a learning example.

## Features

- Responsive, centered login card
- HTML/CSS-only layout with subtle animations
- Basic JavaScript validation for inputs
- Easy to customize styles and behavior

## Demo / Preview
Open `index.html` in your browser to see the login form. The screenshot above shows the default design included in the repository.

## Getting Started

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-46.git

2. Open `index.html` in your browser, or serve the folder with a static server:

   - With Python (3.x): `python -m http.server 8000` and visit `http://localhost:8000`

## Files

- `index.html` — markup for the login form
- `style.css` — styles and responsive rules
- `script.js` — client-side validation and interactivity
- `Screenshot 2024-07-09 081113.png` — preview screenshot used in this README

## Customize

- Edit `style.css` to change colors, spacing, or typography.
- Modify `script.js` to add authentication hooks or connect to an API.
